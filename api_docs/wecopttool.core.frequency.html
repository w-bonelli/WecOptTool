wecopttool.core.frequencyÔÉÅ

frequency(f1, nfreq, zero_freq=True)[source]ÔÉÅ

Construct equally spaced frequency array.

The array includes 0 and has length of nfreq+1. f1 is fundamental frequency (1st harmonic).

Returns the frequency array, e.g., freqs = [0, f1, 2*f1, ..., nfreq*f1].

Parameters

  • f1 (float) ‚Äì Fundamental frequency f1 [Hz].

  • nfreq (int) ‚Äì Number of frequencies.

  • zero_freq (Optional[bool]) ‚Äì Whether to include the zero-frequency.

Return type

ndarray
