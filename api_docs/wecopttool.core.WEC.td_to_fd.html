wecopttool.core.WEC.td_to_fd

WEC.td_to_fd(td, fft=True)[source]

Convert a time-domain array to frequency-domain.

Opposite of wecopttool.core.WEC.fd_to_td().

Calls wecopttool.fd_to_td with the appropriate inputs for the WEC object.

Parameters

  • td (ndarray) – Time-domain real array with shape 2*WEC.nfreq+1 x N for any N.

  • fft (Optional[bool]) – Whether to use the real FFT.

Return type

ndarray

See also

td_to_fd, WEC.fd_to_td
