wecopttool.waves.long_crested_wave

long_crested_wave(efth, direction=0.0)[source]

Create a complex frequency-domain wave elevation from an omnidirectional spectrum.

The omnidirectional spectrum is in the wavespectra format.

Note

The frequencies must be evenly-spaced with spacing equal to the first frequency. This is not always the case when e.g. reading from buoy data. Use interpolation as da.interp(freq=[...]).

Parameters

  • efth (DataArray) – Omnidirection wave spectrum in units of m^2/Hz, in the format used by wavespectra.

  • direction (Optional[float]) – Direction (in degrees) of the long-crested wave.

Return type

DataArray
