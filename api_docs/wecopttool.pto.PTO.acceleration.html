wecopttool.pto.PTO.acceleration

PTO.acceleration(wec, x_wec, x_opt, waves=None, nsubsteps=1)[source]

Calculate the PTO acceleration time-series.

Parameters

  • wec (WEC) – wecopttool.core.WEC object.

  • x_wec (ndarray) – WEC dynamic state.

  • x_opt (ndarray) – Optimization (control) state.

  • waves (Optional[Dataset]) – xarray.Dataset with the structure and elements shown by wecopttool.waves.

  • nsubsteps (Optional[int]) – Number of steps between the default (implied) time steps. A value of 1 corresponds to the default step length.

Return type

ndarray
