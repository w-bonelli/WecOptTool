wecopttool.waves.elevation_fd

elevation_fd(f1, nfreq, directions, amplitudes=None, phases=None, attr=None)[source]

Construct the complex wave elevation xarray.DataArray.

This is the complex wave elevation (m) indexed by radial frequency (rad/s) and wave direction (rad). The coordinate units and names match those from Capytaine.

Parameters

  • f1 (float) – Fundamental frequency f1 [Hz].

  • nfreq (int) – Number of frequencies (not including zero frequency), i.e., freq = [0, f1, 2*f1, ..., nfreq*f1].

  • directions (Union[float, ArrayLike]) – Wave directions in degrees. 1D array.

  • amplitudes (Optional[ArrayLike]) – Wave elevation amplitude in meters.

  • phases (Optional[ArrayLike]) – Wave phases in degrees.

  • attr (Optional[Mapping]) – Additional attributes (metadata) to include in the xarray.DataArray.

Return type

DataArray
