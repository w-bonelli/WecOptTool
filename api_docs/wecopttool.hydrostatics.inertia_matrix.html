wecopttool.hydrostatics.inertia_matrix

inertia_matrix(fb, rho=1025.0, center_of_mass=None, mass=None, rotation_center=None)[source]

Compute the inertia (mass) matrix assuming a constant density for the WEC.

Note

This function assumes a constant density WEC.

Uses capytaine.FloatingBody.compute_rigid_body_inertia on the full mesh.

Parameters

  • fb (FloatingBody) – A capytaine floating body (mesh + DOFs, and optionally center of mass and mass).

  • rho (Optional[float]) – Water density in \(kg/m^3\).

  • center_of_mass (Optional[Iterable[float]]) – Center of gravity/mass.

  • mass (Optional[float]) – Rigid body mass.

  • rotation_center (Optional[Iterable[float]]) – Center of rotation (rx, ry, rz) for hydrostatics calculations.

Raises

  • ValueError – If fb.center_of_mass is not None and center_of_mass is provided with a different value.

  • ValueError – If fb.mass is not None and mass is provided with a different value.

Return type

DataArray
