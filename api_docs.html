API Documentation

The main functionality of WecOptTool is implemented in the core module. Functions and classes in the core module can be accessed directly from WecOptTool as wecopttool.<function> instead of wecopttool.core.<function>. The core module contains:

  • The WEC class

  • Functions for basic functionality

Other functionalities are implemented in additional modules, and can be accessed as wecoptool.<module>.<function>.

Note

The .core should be ommitted when accessing functions and classes in the core submodule. E.g., use wecopttool.WEC instead of wecopttool.core.WEC. For all other modules do include the module name, e.g., wecopttool.waves.regular_wave.

The main way to interact with WecOptTool is through the WEC class.

Modules

wecopttool.core

Core functionality for solving the pseudo-spectral problem for WEC.

wecopttool.pto

Provide power take-off (PTO) forces and produced energy functions for common PTO control approaches.

wecopttool.waves

Provide the wave definition structure and provide functions for creating different common types of waves.

wecopttool.hydrostatics

Functions for calculating hydrostatic and mass properties for floating bodies.

wecopttool.geom

Geometry and parameters for some example devices (WEC).
